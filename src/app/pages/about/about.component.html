About ngFAST What is ngFAST anyways?

Looking for ways to optimize an Angular app I was working on, I realized that the best way was to start from scratch. Not wanting to do everything again if I didn't have to and ngFAST was born.

Built on Angular 8, hosted by Firebase, and using the latest optimizations, ngFAST really is fast. This project scores green on all lighthouse tests, uses preloading hints to ensure the content hits the browser as fast as possible, and serves all resources through Firebase Storage for what is basically a private CDN.
