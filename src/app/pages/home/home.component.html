What is ngFAST? Firebase Angular Simple Template

A high performance PWA template using Firebase Hosting, Angular and Material Design

A publicly usable Angular PWA template that scores ~100 in all Lighthouse tests, depending on the environment it is hosted in.

A full suite of tests provided out of the box ensures the project remains functional as you work on it.

Footnote: Yeah, I know it looks ugly. I am not a great visual designer, but that is up to you! The code is a solid starting point for you to build on.
